Реализованный робот-пылесос обеспечивает следующие функции:
1) Уборка в различых режимах (переключение между режимами автоматическое в зависимости от конфигурации помещения)
2) Возможность изменения скорости уборки
3) Объезд препятствий
4) Звуковое оповещение по окончании уборки, по истечении заряда батареи (если пылесос не нашёл док-станцию) и периодическое оповещение о переполнении пылесборника
5) Автоматический поиск док-станции
6) Ежедневная уборка по заданному времени

Флаги:

Флаги таймера:
01h = переполнение таймера (выставляется прерыванием таймера)
02h = флаг для отсчёта микросекунд (выставляется программистом)
04h = флаг для отсчёта секунд (выставляется программистом)
08h = флаг для начала автоматической уборки по времени (выставляется программистом)
10h - длаг для отсчёта милисекунд (выставляется программистом)

Датчики:
Датчик переполнения пылесборника х 1
УЗ дальномер х 4
Датчик столкновения (бампер) х 4

Прерывания:
1) Нажатие кнопки (любой)
2) Заряд батареи падает ниже 30%
3) Столкновение с препятствием (бампер)
4) Переполнение пылесборника
5 - 8) Получение расстояния до препятствия с дальномеров


Клавиатура 2х3:
1) Начать / закончить уборку
2) Включить / выключить уборку по расписанию
3) Установить минуты (+ для уборки по расписанию)
4) Установить часы (+ для уборки по расписанию)
5) Быстрая / стандартная уборка
6) Включить / выключить звуковые оповещения (можно придумать что-нибудь другое)
